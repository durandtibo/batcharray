# Examples

This directory contains practical examples demonstrating how to use `batcharray` in real-world scenarios.

## Available Examples

### Basic Examples

- **[basic_array_operations.py](basic_array_operations.py)**: Introduction to array module operations
- **[nested_data_structures.py](nested_data_structures.py)**: Working with dictionaries and lists of arrays
- **[masked_arrays.py](masked_arrays.py)**: Handling missing data with masked arrays

### Machine Learning Examples

- **[ml_data_pipeline.py](ml_data_pipeline.py)**: Complete ML data pipeline with batching, shuffling, and splitting
- **[sequence_processing.py](sequence_processing.py)**: Processing time series and sequence data

### Advanced Examples

- **[custom_operations.py](custom_operations.py)**: Creating custom operations with recursive module

## Running Examples

Each example is self-contained and can be run directly:

```bash
cd examples
python basic_array_operations.py
```

## Requirements

Most examples only require the base `batcharray` installation:

```bash
pip install batcharray
```
